# Spark-Orbit
An Augmented Reality interactive book application built with Unity and Vuforia Engine, where physical book pages serve as image targets that trigger 3D planet models and interactive info panels.

✨ Features

- Image Target Detection — each book page is a unique Vuforia image target mapped to a specific planet
- 3D Planet Models — realistic planet models with continuous Y-axis rotation animation
- Pop-Up Animation — smooth scale-up/scale-down animation when planets appear and disappear
- Interactive Info Panel — tap the planet to toggle a Quad-based info panel with planet details
- Pinch to Zoom — pinch gesture support to zoom in/out on the 3D model
- Multi-Target Support — all 8 planets supported simultaneously in a single database
